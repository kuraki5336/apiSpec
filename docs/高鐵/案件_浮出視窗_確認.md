# 案件_浮出視窗_確認
###### tags: `高鐵AI規劃`

>由WEB驗證成功過後的資訊都會夾帶著userID的資料

### 規格

  項目 | 說明
  ---- | ---
  Method | POST
  Required Request Header |  Content-Type:application/json; charset=UTF-8
  測試 環境 API 地址 | `無須定義`
  Real 環境 API 地址 | `無須定義`

### 資料流向
```flow
st=>start: web:>
e=>end: flow:>
st(right)->e(bottom)
```
