# 取得角色

### 作業目的

> 取得角色

## 規格

| 項目                    | 說明                                         |
| ----------------------- | -------------------------------------------- |
| Method                  | GET                                         |
| Required Request Header | Content-Type:application/json; charset=UTF-8 |
| 測試 環境 API 地址      | /vote/role/list                              |
| 通用格式                | 是                                           |
| authorization           | Bearer { token }                             |

```

## 回應

### Response 欄位

| 欄位名稱名稱 | 資料型別 | 長度 | 必須 | 說明     |
| ------------ | -------- | ---- | ---- | -------- |
| roleNo    | string   | -    | Y    | 角色編號 |
| roleName  | string   | -    | Y    | 角色名稱 |


### Response 範例

```json
[
  {
    "roleNo": "A00001",
    "roleName": "紅塵"
  },
  {
    "roleNo": "A00002",
    "roleName": "黃泉"
  },
  {
    "roleNo": "A00003",
    "roleName": "碧落"
  }
]
```
