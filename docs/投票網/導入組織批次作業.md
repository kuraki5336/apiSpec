# 取得 erp 組織資料

### 作業目的

> 修改員工資訊

## 規格

| 項目                    | 說明                                         |
| ----------------------- | -------------------------------------------- |
| Method                  | POST                                         |
| Required Request Header | Content-Type:application/json; charset=UTF-8 |
| 測試 環境 API 地址      | /vote/erp/orglist                            |
| 通用格式                | 是                                           |
| authorization           | Bearer { token }                             |

## 回應

> 簡單的回應訊息，來確定是否存檔成功
> 使用通用格式 code 200 msg succese

```json
{
  "state": "200",
  "msg": "succese",
  "error": "",
  "data": {}
}
```
