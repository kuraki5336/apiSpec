# 投票資料刪除作業

### 作業目的

> 投票資料刪除作業

## 規格

| 項目                    | 說明                                         |
| ----------------------- | -------------------------------------------- |
| Method                  | POST                                         |
| Required Request Header | Content-Type:application/json; charset=UTF-8 |
| 測試 環境 API 地址      | /vote/order/delete                           |
| 通用格式                | 是                                           |

## 請求

### Request 欄位

| 欄位名稱 | 資料型別 | 長度 | 必須 | 說明                  |
| -------- | -------- | ---- | ---- | --------------------- |
| act_no   | string   | 64   | Y    | form.act_no(此單單號) |

### Request 範例

```json
{
  "act_no": "20200730000000A0001"
}
```

## 回應

> 簡單的回應訊息，來確定是否存檔成功
> 使用通用格式 code 0 msg succese
